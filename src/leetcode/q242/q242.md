# 242 有效的字母异位词

## 1. 题目

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

示例1:
```
输入: s = "anagram", t = "nagaram"
输出: true
```
示例 2:
```
输入: s = "rat", t = "car"
输出: false
```
说明:

你可以假设字符串只包含小写字母。

进阶:

如果输入字符串包含 unicode 字符怎么办？你能否调整你的解法来应对这种情况？

## 2.解题思路

### 2.1 哈希表

1. 哈希表统计s中字符个数，结果为m
2. 遍历t字符串中对每个字符，削减m中对应字符的个数
3. 考察m中每个字符的个数是否为0

- 时间复杂度:O(n)
- 空间复杂度:O(n)

### 2.2 排序

对两个字符串进行排序，然后比较。

- 时间复杂度:O(nlogn~n2)
- 空间复杂度:O(n)
